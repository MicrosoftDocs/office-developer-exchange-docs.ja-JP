## <a name="working-with-identifiers"></a><span data-ttu-id="591a8-161">識別子の処理</span><span class="sxs-lookup"><span data-stu-id="591a8-161">Working with identifiers</span></span>

Consider the following when you develop your EWS Managed API or EWS application:</span></span>
