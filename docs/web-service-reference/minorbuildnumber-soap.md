# <a name="minorbuildnumber-soap"></a>MinorBuildNumber/(SOAP)

**Minorbuildnumber**要素は、サーバーのマイナービルド番号を表します。 
  
```XML
<MinorBuildNumber/>
```

 **int**
## <a name="attributes-and-elements"></a>属性と要素

以下のセクションで、属性、子要素、親要素について説明します。
  
### <a name="attributes"></a>属性

なし。
  
### <a name="child-elements"></a>子要素

なし。
  
### <a name="parent-elements"></a>親要素

|**要素**|**説明**|
|:-----|:-----|
|[ServerVersionInfo (SOAP)](serverversioninfo-soap.md) <br/> |要求を処理したサーバーのバージョンが保存されています。  <br/> |
   
## <a name="text-value"></a>テキスト値

要求を処理したサーバーのマイナービルド番号を表す整数値。
