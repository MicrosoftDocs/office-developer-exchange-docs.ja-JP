# <a name="workingperiod"></a>WorkingPeriod

**WorkingPeriod**要素には、日、およびメールボックス ユーザーの時間の作業週が含まれています。

[FreeBusyView](freebusyview.md)

[WorkingPeriod](workingperiod.md)
  
```xml
<WorkingPeriod>
   <DayOfWeek>...</DayOfWeek>
   <StartTimeInMinutes>...</StartTimeInMinutes>
   <EndTimeInMinutes>...</EndTimeInMinutes>
</WorkingPeriod>
```

 **WorkingPeriod**
## <a name="attributes-and-elements"></a>属性および要素

以下のセクションで、属性、子要素、親要素について説明します。
  
### <a name="attributes"></a>属性

なし。
  
### <a name="child-elements"></a>子要素

|**要素**|**説明**|
|:-----|:-----|
|[DayOfWeek (WorkingPeriod)](dayofweek-workingperiod.md) <br/> |メールボックス ユーザーのスケジュールの稼働日のリストが含まれています。  <br/> |
|[StartTimeInMinutes](starttimeinminutes.md) <br/> |メールボックスがユーザーの稼働日の開始を表します。  <br/> |
|[EndTimeInMinutes](endtimeinminutes.md) <br/> |メールボックスがユーザーの稼働日の終了を表します。  <br/> |
   
### <a name="parent-elements"></a>親要素

|**要素**|**説明**|
|:-----|:-----|
|[WorkingPeriodArray](workingperiodarray.md) <br/> |メールボックス ユーザーについて、期間の作業が含まれています。  <br/> この要素への XPath 式は、次のようにします。  <br/>  `/GetUserAvailabilityResponse/FreeBusyResponseArray/FreeBusyResponse/FreeBusyView/WorkingHours/WorkingPeriodArray` <br/> |
   
## <a name="remarks"></a>備考

発生したシーケンスは、すべての子要素の一覧です。 MicrosoftExchange Server 2007 がインストールされているクライアント アクセス サーバーの役割を実行しているコンピューターの EWS 仮想ディレクトリには、この要素を記述するスキーマがあります。
