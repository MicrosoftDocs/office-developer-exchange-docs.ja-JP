# <a name="getorganizationrelationshipsettingsrequestmessage-soap"></a>GetOrganizationRelationshipSettingsRequestMessage (SOAP)

**GetOrganizationRelationshipSettingRequestMessage**要素は、 [GetOrganizationRelationshipSettings 操作 (SOAP)](getorganizationrelationshipsettings-operation-soap.md)操作要求を表します。 **GetOrganizationRelationshipSettingRequestMessage**要素は、内部使用のみ。 クライアントでは、この要素は使用されません。 
  
```XML
<GetOrganizationRelationshipSettingRequestMessage>
   <Request/>
</GetOrganizationRelationshipSettingRequestMessage>
```

 **GetOrganizationRelationshipSettingsRequest**

なし。
