# <a name="mailboxmoveenabled-soap"></a>MailboxMoveEnabled (SOAP)

**MailBoxMoveEnabled**要素は、 **MailBoxMoveEnabled ()** フラグを表します。 **MailBoxMoveEnabled**要素は、内部使用のみを対象としています。 この要素はクライアントによって使用されません。 
  
```XML
<MailBoxMoveEnabled>true | false</MailBoxMoveEnabled>
```

**Boolean**
