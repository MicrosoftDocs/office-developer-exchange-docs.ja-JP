# <a name="displayname-pox"></a>DisplayName (POX)

**DisplayName**要素は、ユーザーの表示名を表します。 
  
- [(POX) を自動検出](autodiscover-pox.md) 
- [応答 (POX)](response-pox.md) 
- [ユーザー (POX)](user-pox.md) 
- [DisplayName (POX)](displayname-pox.md)
  
```xml
<DisplayName/>
```

なし。
