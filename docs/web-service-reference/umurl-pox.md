**UMUrl**要素は、最適なメールが有効なユーザーのユニファイド メッセージングの Web サービスのインスタンスの URL を指定します。

なし。
  
### <a name="child-elements"></a>子要素

|**要素**|**説明**|

## <a name="text-value"></a>テキスト値

テキスト値は、ユニファイド メッセージング サーバーの役割が Exchange トポロジの場合、インストールされている Exchange サーバーの URL を指定します。
