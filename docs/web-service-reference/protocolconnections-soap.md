# <a name="protocolconnections-soap"></a>ProtocolConnections (SOAP)

**Protocolconnections**要素には、0個以上のプロトコル接続が含まれています。 
  
```XML
<ProtocolConnections>
   <ProtocolConnection/>
</ProtocolConnections>
```
