# <a name="protocolconnections-soap"></a>ProtocolConnections (SOAP)

**ProtocolConnections**要素には、0 個以上のプロトコルの接続が含まれています。 
  
```XML
<ProtocolConnections>
   <ProtocolConnection/>
</ProtocolConnections>
```
