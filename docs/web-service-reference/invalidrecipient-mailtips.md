# <a name="invalidrecipient-mailtips"></a>InvalidRecipient (メール ヒント)

**InvalidRecipient**要素は、受信者が有効ではないかどうかを示します。 
  
```XML
<InvalidRecipient>true | false</InvalidRecipient>
```

 **ブール型 (Boolean)**
## <a name="attributes-and-elements"></a>属性および要素

以下のセクションで、属性、子要素、親要素について説明します。
  
### <a name="attributes"></a>属性

なし。
  
### <a name="child-elements"></a>子要素

なし。
  
### <a name="parent-elements"></a>親要素

|**要素**|**説明**|
|:-----|:-----|
|[メール ヒント](mailtips.md) <br/> |さまざまな種類のメール ヒントの値を表します。  <br/> |
   
## <a name="text-value"></a>テキスト値

**受信者が有効ではない場合は、この要素のテキスト値は** 受信者が無効な場合に、値は**false です**。 
  
## <a name="remarks"></a>備考

この要素を記述するスキーマは、Exchange Web サービスをホストする IIS 仮想ディレクトリに置かれています。
