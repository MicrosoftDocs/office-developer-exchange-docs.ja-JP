# <a name="arrayofuserresponse-soap"></a>ArrayOfUserResponse (SOAP)

**Arrayofuserresponse**要素には、 [USERRESPONSE (SOAP)](userresponse-soap.md)要素の配列が含まれています。 
  
```XML
<ArrayOfUserResponse>
   <UserResponse/>
</ArrayOfUserResponse>
```
