# <a name="arrayofuserresponse-soap"></a>ArrayOfUserResponse (SOAP)

**ArrayOfUserResponse**要素には、 [UserResponse (SOAP)](userresponse-soap.md)の要素の配列が含まれています。 
  
```XML
<ArrayOfUserResponse>
   <UserResponse/>
</ArrayOfUserResponse>
```

 **ArrayOfUserResponse**
