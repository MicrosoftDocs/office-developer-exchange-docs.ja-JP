# <a name="setoofstatus-um-web-service"></a>SetOofStatus (UM web サービス)

**Setoofstatus**要素は、要求を行うユーザーのユニファイドメッセージング不在 (OOF) 状態を設定するための要求を定義します。 
  
[SetOofStatus (UM web サービス)](setoofstatus-um-web-service.md)
  
```xml
<SetOofStatus>
  <status/>
</SetOofStatus>
```
