# <a name="domain-getfederationinformation-soap"></a>ドメイン (GetFederationInformation) (SOAP)

**ドメイン**要素は、フェデレーションの信頼されているドメインを識別します。 
  
```XML
<Domain/>
```

 **string**
## <a name="attributes-and-elements"></a>属性および要素

以下のセクションで、属性、子要素、親要素について説明します。
  
### <a name="attributes"></a>属性

なし。
  
### <a name="child-elements"></a>子要素

なし。
  
### <a name="parent-elements"></a>親要素

|**要素**|**説明**|
|:-----|:-----|
|[要求 (GetFederationInformation) (SOAP)](request-getfederationinformationsoap.md) <br/> |[GetFederationInformation 操作 (SOAP)](getfederationinformation-operation-soap.md)要求を表します。  <br/> |
   
## <a name="text-value"></a>テキスト値

テキスト値は、フェデレーションの信頼を含むドメインのドメイン名を表します。
