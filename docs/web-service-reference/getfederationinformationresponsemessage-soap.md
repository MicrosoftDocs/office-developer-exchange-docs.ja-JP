# <a name="getfederationinformationresponsemessage-soap"></a>GetFederationInformationResponseMessage (SOAP)

**GetFederationInformationResponseMessage**要素は、 [GetFederationInformation 操作 (SOAP)](getfederationinformation-operation-soap.md)要求に対する応答を定義します。 
  
```XML
<GetFederationInformationResponseMessage>
   <Response/>
</GetFederationInformationResponseMessage>
```

 **GetFederationInformationResponseMessage**

なし。
