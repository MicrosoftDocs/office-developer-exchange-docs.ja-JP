# <a name="unknownattendeeconflictdata"></a>UnknownAttendeeConflictData

**UnknownAttendeeConflictData**要素は、ユーザー、配布リスト、または連絡先ではない、未解決の出席者または出席者を表します。

## <a name="remarks"></a>注釈

Active Directory ディレクトリサービスオブジェクトに対して解決できない場合、出席者は不明です。 ユーザー、グループ、または連絡先であると判断できない場合、出席者は未解決になります。 たとえば、メールが有効なパブリックフォルダーである場合、出席者は解決されません。
  
この要素を記述するスキーマは、クライアントアクセスサーバーの役割がインストールされている Microsoft Exchange Server 2007 を実行しているコンピューターの EWS 仮想ディレクトリにあります。
