# <a name="usersettings-soap"></a>ユーザー (SOAP)

**ユーザー**の要素は、ユーザー設定のコレクションを表します。 
  
```XML
<UserSettings>
    <UserSetting/>
</UserSettings>
```

 **ユーザー**

### <a name="attributes"></a>属性

なし。

|**要素**|**説明**|
|:-----|:-----|
|[UserSetting (SOAP)](usersetting-soap.md) <br/> |1 つのユーザー設定を表します。  <br/> |
