[エラー (POX)](error-pox.md)
  
[エラー コード (POX)](errorcode-pox.md)


### <a name="attributes"></a>属性

なし。
  
### <a name="child-elements"></a>子要素

なし。

|**要素**|**説明**|
|:-----|:-----|
|[エラー (POX)](error-pox.md) <br/> |自動検出エラー応答が含まれています。  <br/> |
