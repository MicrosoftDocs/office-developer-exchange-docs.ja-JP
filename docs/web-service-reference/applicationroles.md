# <a name="applicationroles"></a>ApplicationRoles

**Applicationroles**要素は、呼び出し元パートナーアプリケーションが現在の呼び出しに対して使用するアプリケーションロールを指定します。 
  
```XML
<ApplicationRoles>
    <Role></Role>
</ApplicationRoles>
```

 **非 Emptyarrayofroletype**
## <a name="attributes-and-elements"></a>属性と要素

以下のセクションで、属性、子要素、親要素について説明します。
  
### <a name="attributes"></a>属性

なし。
  
### <a name="child-elements"></a>子要素

|**Element**|**説明**|
|:-----|:-----|
|[役割](role.md) <br/> |管理役割を表す文字列を指定します。  <br/> |
   
### <a name="parent-elements"></a>親要素

|**要素**|**説明**|
|:-----|:-----|
|[Get-managementrole](managementrole.md) <br/> |管理役割を指定します。  <br/> |
   
## <a name="remarks"></a>注釈

この要素は Exchange Server 2013 で導入されました。
  
この要素を記述するスキーマは、Exchange Web サービスをホストする IIS 仮想ディレクトリに置かれています。
