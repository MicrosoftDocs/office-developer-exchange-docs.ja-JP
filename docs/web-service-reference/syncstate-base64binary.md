# <a name="syncstate-base64binary"></a>同期状態 (base64Binary)

**同期状態**の要素は、スレッドの同期の状態を指定します。 
  
```XML
<SyncState></SyncState>
```

 **base64Binary**
## <a name="attributes-and-elements"></a>属性および要素

以下のセクションで、属性、子要素、親要素について説明します。
  
### <a name="attributes"></a>属性

なし。
  
### <a name="child-elements"></a>子要素

なし。
  
### <a name="parent-elements"></a>親要素

[会話 (ConversationRequestType)](conversation-conversationrequesttype.md) | [(ConversationResponseType) の会話](conversation-conversationresponsetype.md)
  
## <a name="text-value"></a>テキスト値

**同期状態**の要素のテキスト値は、会話の同期状態です。 
  
## <a name="remarks"></a>備考

この要素は Exchange Server 2013 で導入されました。
  
この要素を記述するスキーマは、Exchange Web サービスをホストする IIS 仮想ディレクトリに置かれています。
